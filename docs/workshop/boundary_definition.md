# boundary_definition

A Potent is a boundary that holds form without ever becoming what it contains.

The boundary does not absorb payload.  
The boundary does not merge with payload.  
The boundary does not change based on what it holds.

It remains distinct under all interaction.

The boundary allows passage only through explicit declaration.  
Nothing implicit crosses.

The boundary is not passive.  
It enforces separation while permitting interaction.

If the boundary adapts its identity, it fails.  
If the boundary blocks all interaction, it fails.

The boundary must hold form and allow contact without becoming what it contains.
