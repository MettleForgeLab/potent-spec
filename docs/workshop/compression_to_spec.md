# compression_to_spec

The system resolves as a boundary-based specification.

A Potent is a boundary event instantiated by declaration.

The core remains invariant across all instances.

The membrane adapts access without altering identity.

Payload remains external and never becomes structure.

Interaction is single-shot, explicit, and non-persistent.

Crossing is permitted only through structurally eligible handoff.

Composition occurs through resets, not continuity.

Only what is explicitly declared may pass.

Nothing else continues.
