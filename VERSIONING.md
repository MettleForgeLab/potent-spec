\# Versioning



This repository follows semantic versioning:



MAJOR.MINOR.PATCH



\## Principles



\- MAJOR versions change the structure of the specification

\- MINOR versions add non-breaking structural elements

\- PATCH versions correct or clarify without altering structure



\## Constraints



\- The canonical surface (`SPECIFICATION.md`) must not change without a version increment

\- No silent modification is permitted

\- All changes must be explicitly recorded in `CHANGELOG.md`



\## Stability



Version `v1.0.0` is considered a stable canonical baseline.
